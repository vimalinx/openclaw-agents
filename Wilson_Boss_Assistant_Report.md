- 时效: 10 分钟内完成

**演示脚本：**

```python
# 自动发布流程
async def auto_post(topic: str):
    # 1. 爬取热点数据
    hot_data = await crawl_hot_topic(topic)

    # 2. 生成策略
    strategy = analyze_hot_data(hot_data)

    # 3. 生成内容
    content = generate_content(strategy, topic)

    # 4. 生成配图
    images = generate_images(content, style="xiaohongshu")

    # 5. 自动发布
    await publisher.publish(content, images, tags=strategy.tags)

    return {"status": "success", "post_url": "..."}
```
